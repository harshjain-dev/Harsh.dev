### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/harshjain2094/Harsh.dev.git
    cd Harsh.dev
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:3000`.
